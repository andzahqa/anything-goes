# Phase 1: Foundation Projects

This folder contains your first projects as you learn the fundamentals of web development.

## Projects

### Project 1: Personal Dashboard (Weeks 1-2)
**Location**: `project-1-personal-dashboard/`  
**Skills**: HTML5, CSS3, JavaScript basics, Flexbox/Grid, Animations  
**Goal**: Build a responsive personal dashboard with interactive cards

### Project 2: Data Fetcher Dashboard (Weeks 3-4)
**Location**: `project-2-data-fetcher/`  
**Skills**: Fetch API, Async/Await, Error handling, Working with real APIs  
**Goal**: Enhance dashboard with real data from public APIs

## Getting Started

1. Create a new folder for each project
2. Start with an `index.html` file
3. Add CSS and JavaScript files
4. Commit your progress with Git
5. Have fun and experiment!

## Tips

- Focus on making it work first, then make it pretty
- Use browser DevTools to debug
- Don't copy/paste - type everything yourself to build muscle memory
- Take breaks when stuck - sometimes stepping away helps
