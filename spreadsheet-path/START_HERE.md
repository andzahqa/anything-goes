2. Click "Blank" to create new sheet
3. Type these formulas:
   ```
   =SUM(A1:A5)
   =AVERAGE(B1:B5)
   =IF(C1>100, "High", "Low")
   ```
4. Fill some numbers in rows 1-5
5. See the formulas work!
